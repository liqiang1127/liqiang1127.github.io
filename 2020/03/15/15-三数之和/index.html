15.三数之和

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

传送门

定数组 nums = [-1, 0, 1, 2, -1, -4]，
满足要求的三元组集合为：

[[-1, 0, 1],
[-1, -1, 2]]

分析一下

这题挺简单的，先排个序。固定一个数字，然后用双指针分别从前后找组合。

考点：双指针

重点：去重

三个地方去重

  1. 固定的数字（也就是代码中的a）如果这个和前面考虑过的元素一样，那么直接跳过。因为在考虑之前的元素的时候，肯定已经把当前元素的情况考虑过了。

  2. 当找到一个组合的时候，l或者r对应的数字如果和之前考虑过的情况一样，那也跳过。此时，a确定，如果l再和之前的一样，那r肯定也一样了，毕竟a+b+c=0，两个都确定了，第三个自然也确定了，那么找到的答案必然是重复答案。
